My Future

ผมอยากทํางานเป็น Frontend developer
Frontend Developer คือผู้พัฒนาเว็บไซต์หรือแอปพลิเคชันที่รับผิดชอบในส่วนของการออกแบบและพัฒนาองค์ประกอบที่ผู้ใช้เห็นและโต้ตอบได้ เช่น รูปแบบเว็บไซต์, ปุ่ม, เมนู, ฟอร์ม, และการจัดวางเนื้อหา โดยใช้เทคโนโลยีหลัก ๆ คือ HTML, CSS และ JavaScript เพื่อทำให้เว็บดูสวยงามและใช้งานได้ดีบนทุกอุปกรณ์ รวมถึงการเชื่อมต่อกับ Backend เพื่อแสดงข้อมูลจากเซิร์ฟเวอร์ให้กับผู้ใช้
เพราะการพัฒนาทักษะและสร้างโอกาสในอาชีพ: Frontend เป็นจุดเริ่มต้นที่ดีในการขยายไปสู่
Full-stack หรือ UI/UX Design ซึ่งเปิดโอกาสการเติบโตในสายงานเพิ่มเติม
เเละทํางานจากที่ไหนก็ได้ส่วนนี้ที่ผมชอบเเละมีความต้องการในตลาดสูง
อยากเก็บเงินให้ได้เยอะๆเเล้วไปเที่ยวต่างประเทศ
